# VueAiJustPlay

- 디자인 시스템이 있는 상황에서, AI 를 사용하여, UI 를 구성하는 방법을 연습합니다.
- /missions 폴더에 미션이 있습니다. 각자 사용하는 AI 를 활용하여, 미션을 수행해 보세요

## 실행 요구사항 
- Node.js v22.11.0

## 의존성 설치 방법

```sh
npm install
```

### 개발모드 실행 (소스 수정시 자동반영 기능)

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
